# Оно тебе надо

Аукцион вещей, в которые никто не верил

## О проекте

Проект "Оно тебе надо" представляет собой лендинг для аукциона редких и необычных вещей. Здесь собраны предметы с историей, в которые когда-то никто не верил, но теперь они представляют культурную и историческую ценность.

## Технологии

- HTML5
- CSS3
- Адаптивная верстка
- Flexbox/Grid верстка

## Структура проекта

- `index.html` - главная страница
- `style.css` - файл стилей
- `/images` - директория с изображениями
- `/fonts` - директория со шрифтами

## Начало работы

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
