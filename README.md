# Оно тебе надо

Аукцион вещей, в которые никто не верил

## О проекте

Проект "Оно тебе надо" представляет собой лендинг для аукциона редких и необычных вещей.

## Технологии

- HTML5
- CSS3
- Адаптивная верстка

## Структура проекта

- `index.html` - главная страница
- `/styles` - директория с CSS файлами
  - `global.css` - базовые стили
  - `style.css` - основные стили проекта
- `/images` - директория с изображениями
- `/fonts` - директория со шрифтами

## Начало работы

1. Клонируйте репозиторий
2. Откройте `index.html` в браузере
